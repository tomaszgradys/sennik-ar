# hulm.pro - brand guide pack

Paczka dla lokalnego wariantu sennik.tv pod domeną `hulm.pro`.

## Co jest w środku

- `BRAND_GUIDE.md` - pełny guide w Markdown.
- `HulmPro_BrandGuide.pdf` - wersja do czytania i wysyłki.
- `HulmPro_BrandGuide.docx` - wersja edytowalna.
- `assets/logo/` - logo, mark, wordmark, OG image.
- `assets/favicon/` - favicon SVG i PNG 512.
- `assets/tokens/design-tokens.json` - tokeny design/dev.
- `assets/css/hulm-theme.css` oraz `hulm-components.css` - starter CSS.
- `assets/html/` - statyczne preview night/day.
- `assets/copy/` - lokalne teksty UI i SEO przykłady.
- `assets/illustrations/` - mini zestaw ilustracji w stylu sennik.tv.
- `ui/mockups/` - PNG makiet homepage/blog.
- `implementation/` - gotowe fragmenty HTML.

Fontów nie dołączam do paczki. Guide wskazuje tylko rekomendowany stack.
